# Ecumarkett Wiki

Ecumarkett es una aplicacion de escritorio hecha en Java para gestionar productos, categorias, clientes, proveedores, entradas, salidas y reportes.

Este proyecto empezo como una idea de aprendizaje, paso por varias pruebas y quedo ordenado como una base clara para mostrar, estudiar o seguir mejorando.

## Resumen rapido

1. La app usa MySQL local con XAMPP.
2. Los formularios y pantallas estan hechos en NetBeans.
3. Los reportes se generan con JasperReports.
4. La documentacion esta pensada para que cualquiera entienda el proyecto sin perderse.
